Tweeply | Automatic Twitter Replies

A cognitive application to send custom replies to tweets

Tweeply is a cognitive Bluemix-based Web application that sends custom replies to tweets sent to a particular Twitter handle.

This application is most suited for automated Twitter-based customer care services, election campaigns and almost anything interactive!

In this version, Tweeply handles customer care queries and sends out suitable replies for the same.

Tweeply uses Node-RED and the Twitter API to analyze all public tweets made to a specific handle, and the Natural Language Classifier Watson API to classify them.
